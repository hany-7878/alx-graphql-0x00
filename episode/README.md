# Episode Query Task

This directory contains GraphQL queries for retrieving episode details using the episode(id: ID!) field.

## Endpoint
https://rickandmortyapi.com/graphql

## Files Included

- episode-page-1.graphql
- characters-page-1-output.json
- characters-page-2.graphql
- characters-page-2-output.json
- characters-page-3.graphql
- characters-page-3-output.json
- characters-page-4.graphql
- characters-page-4-output.json

## How to Use

1. Open the GraphQL endpoint in GraphiQL.
2. Copy the contents of any .graphql file.
3. Run the query.
4. Copy the JSON response and save it in the matching -output.json file.
